## pre-launch:
1. ```composer install --no-dev --optimize-autoloader```
2. ```php app/console doctrine:schema:create```
3. ```cd web```
4. ```npm install``` or ```yarn install```
5. ```npm update``` or ```yarn update```
6. Congrats - app ready to launch
